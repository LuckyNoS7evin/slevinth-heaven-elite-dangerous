# Slevinth Heaven — Elite Dangerous

Live site: https://elite.sweetffa.com

Summary

Slevinth Heaven — Elite Dangerous includes a Blazor web application, a companion backend API, and a Windows desktop application. The repository contains the web client, server components, and the desktop app — all built on .NET (Blazor frontend and ASP.NET Core API for the web; the desktop client is a .NET Windows application). The project provides browser-based interfaces, server endpoints, and a desktop client used to support features for the Elite Dangerous community and tooling.

Key points

- Live URL: https://elite.sweetffa.com
- Built with Blazor, ASP.NET Core, and .NET desktop technologies
- Contains frontend, backend, and Windows desktop projects plus configuration files for local development and deployment

See the project folders and configuration files in the repository for development and deployment instructions.
